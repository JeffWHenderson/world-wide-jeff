# World Wide Jeff

A free, open-source language learning app built around spaced repetition, stories, and picture lessons. No accounts, no ads, no tracking — progress lives in your browser.

**The app lives in `apps/frontend`.** That's the only thing that matters. The `apps/language-app` folder is an old Next.js experiment, ignore it.

## Quick start

```bash
cd apps/frontend
yarn          # install dependencies
yarn dev      # start dev server at http://localhost:5173
```
